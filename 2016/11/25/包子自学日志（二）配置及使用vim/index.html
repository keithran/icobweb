vim常用操作

对于我来说，通过vim给我带来很多便利，这些便利就是源于操作的快捷性。

认清两个模式

Insert：插入模式，这个时候你可以coding了。
Normal：标准模式，对你的code进行编辑操作。
我们最需要掌握的知识点就是在Normal模式下的快捷操作。

基本操作

i → Insert 模式，按 ESC 回到 Normal 模式.
x → 删当前光标所在的一个字符。
:wq → 存盘 + 退出 (:w 存盘, :q 退出)
dd → 删除当前行，并把删除的行存到剪贴板里
p → 粘贴剪贴板

插入命令

a → 在光标后插入
o → 在当前行后插入一个新行
O → 在当前行前插入一个新行
cw → 替换从光标所在位置后到一个单词结尾的字符

移动光标

0 → 数字零，到行头
^ → 到本行第一个不是blank字符的位置（所谓blank字符就是空格，tab，换行，回车等）
g_ → 到本行最后一个不是blank字符的位置。
/pattern → 搜索 pattern 的字符串

拷贝／粘贴

P → 粘贴
yy → 拷贝当前行当行于 ddP

Undo/Redo（撤销／前进）

u → undo
Ctrl+r → redo

打开/保存/退出/改变文件

:e → 打开一个文件
:w → 存盘
:saveas → 另存为
:wq → 保存并退出
:!q → 退出不保存 :!qa 强行退出所有的正在编辑的文件。
:bn 和 :bp → 你可以同时打开很多文件，使用这两个命令来切换下一个或上一个文件。

高级操作1-重复操作

. (小数点) → 可以重复上一次的命令
N → 重复某个命令N次

高级操作2-移动光标

NG → 到第 N 行
gg → 到第一行。
G → 到最后一行。
按单词移动：

  1. w → 到下一个单词的开头
  2. e → 到下一个单词的结尾

高级操作3-位置搜索

fa → 到下一个为a的字符处，你也可以fs到下一个为s的字符。
t, → 到逗号前的第一个字符。逗号可以变成其它字符。
3fa → 在当前行查找第三个出现的a。
F 和 T → 和 f 和 t 一样，只不过是相反方向。
Tip: dt” → 删除所有内容直到遇到双引号“

va)：选择同级别括号内的内容，包含括号

vi)：选择同级别括号内的内容，不含括号

高级操作4-块操作

块操作，典型的操作：

实例应用-1：利用C-V进行行位缩进

J → 把所有的行连接起来（变成一行）
< 或 > → 左右缩进
= → 自动给缩进

实例应有-2：所有被选行后添加

Ctrl+v → 选中相关的行

操作录制-我还没用过

其实就是将操作过程录制成一个宏，需要的时候进行调用就可以了。

qa：把操作记录在寄存器a上，当然a也可以换成其他的字母。
@a：调用录制的宏。
@@：快捷调用最新录制的宏。
N@a/N@@：N是随意数字，调用某个宏N次。
